
Add folder called 'Codes' to the FilePath on MATLAB. Codes contains:
1. Main_CorrelationAnalysis: Please set the correct filepaths and run the code.
2. ConvertToD package is called by Main_CorrelationAnalysis and is used to convert PWS(RMS Intensity data) to D-map which has values between 0-3.
2. MatchPWS_Confocal function is called by Main_CorrelationAnalysis and is used to find slices in confocal with max correlation with PWS.
